## Tuumik MCP

This repository contains Tuumik MCP, an application that enables AI tools to communicate with Tuumik and is meant to run alongside it. Tuumik is a self-hosted team management platform that provides tools to make in-office and remote work more efficient.

Tuumik's repository is at https://github.com/tuumiksystems/tuumik

Tuumik's website is at https://www.tuumik.com

## Documentation

You can find documentation on how to deploy and customize Tuumik MCP in [Tuumik's repository](https://github.com/tuumiksystems/tuumik).
